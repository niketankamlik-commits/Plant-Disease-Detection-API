API Reference

Build powerful plant analysis apps with the Phyto-Scan engine.

Authentication

Phyto-Scan uses API Keys to authenticate requests. You can view and manage your keys in your dashboard. All API requests must be made over HTTPS.

Authentication is performed by passing a X-API-KEY header.

X-API-KEY: YOUR_API_KEY_HERE

Disease Prediction

POST /api/predict

Analyze a plant leaf image and receive detection results.

Parameters

  • file (Form Data) - The image file (JPEG, PNG, WEBP)

Example with Python

import requests url = "http://127.0.0.1:8000/api/predict" files = {'file': open('leaf.jpg', 'rb')} headers = {'X-API-KEY': 'your_secret_key'} response = requests.post(url, files=files, headers=headers) print(response.json())

Successful Response

{ "success": true, "disease_name": "Tomato - Late Blight", "confidence": 98.45, "is_healthy": false, "recommendation": "Isolate the plant..." }
